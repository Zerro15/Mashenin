# Product Outline

## Positioning

mashenin is a Discord-style voice-first workspace for ongoing communities, teams, and always-on rooms.

## Core User Flow

1. Open app
2. See who is online and in voice
3. Join a room in one tap
4. Drop messages, links, memes, or event plans in room chat

## MVP Screens

- Home
- Room
- Friends
- Events
- Admin

## MVP Features

- invite-only access
- member presence
- permanent rooms
- ad-hoc rooms
- room text chat
- event RSVP

## Non-Goals For MVP

- open public discovery
- large-scale moderation
- native mobile apps
- end-to-end encryption
